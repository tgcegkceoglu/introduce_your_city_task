# Proje Açıklaması

Bu proje, Denizli şehrini tanıtmak amacıyla geliştirilmiş bir mobil uygulamadır. Uygulama, şehirdeki gezilecek yerleri, geleneksel mutfak,  ve daha birçok bilgiyi kullanıcılara sunmayı hedefler.
